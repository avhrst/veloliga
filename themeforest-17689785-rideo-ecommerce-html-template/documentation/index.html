Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside rideo folder.

Pages

    • index.html is for Default Home page
    • index-2.html is for Home page two
    • index-3.html is for Home page three
    • shop.html is for Shop page
    • shop-gird-left.html is for Shop grid left sidebar page
    • shop-gird-right.html is for Shop grid right sidebar page
    • shop-list-left.html is for Shop list left sidebar page
    • shop-list-right.html is for Shop list right sidebar page
    • product-details.html is for Product details page
    • cart.html is for Cart page
    • wishlist.html is for Wishlist page
    • checkout.html is for Checkout page
    • login.html is for Login page
    • my-account.html is for My Account page
    • blog-1.html is for Blog page style 1
    • blog-2.html is for Blog page style 2
    • single-blog.html is for Blog-details page
    • error-404.html is for 404 Error page
    • about.html is for About Us page
    • contact.html is for Contact page
